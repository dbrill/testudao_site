## Stack
`code`: React with typescript<br />
`env`: Github pages

### `npm run deploy`

Builds sources and deploys them to github pages

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
